⚠️ Lưu ý quan trọng

  • GND của ESP32, L298N, nguồn 12V phải nối chung (common ground)
  • Tụ 100–220uF lắp song song hai đầu quạt DC (chống nhiễu)
  • Relay module dùng transistor — GPIO 3.3V điều khiển được
  • AS608 dùng 3.3V, KHÔNG dùng 5V sẽ hỏng chip
  • Servo rút thêm dòng khi tải nặng — nên dùng nguồn riêng
  • Không dùng GPIO 6–11 (kết nối flash nội)

🖥️ Cài đặt Mosquitto Broker

  • Cài Mosquitto: apt install mosquitto
  • Thêm vào mosquitto.conf:
  • listener 1883
  • listener 9001
  • protocol websockets
  • allow_anonymous true
  • Khởi động: systemctl start mosquitto
  • Nhấn ⚙️ trong web → nhập IP máy broker
